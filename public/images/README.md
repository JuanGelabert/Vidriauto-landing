# Imágenes del Sitio

## Hero Section

Coloca aquí la imagen del almacén/warehouse para el Hero:

- **Archivo**: `hero.jpg` (o `.png`, `.webp`)
- **Recomendaciones**:
  - Resolución: Mínimo 1920x1080px (Full HD)
  - Formato: JPG (alta calidad) o WebP (mejor compresión)
  - Contenido: Almacén con estantes de vidrios de autos
  - La imagen se aplicará con blur en el fondo del Hero
